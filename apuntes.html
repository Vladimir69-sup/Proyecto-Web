Etiquetas HTML
Etiqueta Para qué sirve Ejemplo mínimo
navSe utiliza para crear una interfaz de de navegación 
<nav>
    <ul>
        <li><a href="index.html">Inicio</a></li>
        <li><a href="#acerca">Acerca de</a></li>
        <li><a href="#talleres">Talleres</a></li>
        <li><a href="contacto.html">Contacto</a></li>
        <li><a href="#quimica">Quimica</li>
    </ul>
</nav>
preSe utilia para mandar un párrafo o grupo de texto tal cual lo escribimos en el código 
<pre><code>
        #include <stdio.h<

        int main ()
        {
            printf("Hola Tú");
            return 0;
        }
olSe utiliza para la creación de listas enumeradas de acuerdo al orden de escritura de los elementos 
<ol>    
    <li>Elemento 1.1</li>
    <li>Elemento 2.1</li>
    <li>Elemento 3.1</li>
</ol>
ulSe utiliza para la creación de listas desordenads, solo se ocupan viñetas u listaje que no implica jerarquía 
<ul>
    <li>Lunes: 10:20 - 12:00</li>
    <li>Miércoles: 8:40 - 10:20</li>
    <li>Viernes: 8:40 - 9:30</li>
</ul>
pSe utiliza para la escritura de líneas de texto dentro de tu página web 
<p>Utilizamos la batidora para preparar la masa.<p>
brSe utiliza para realizar saltos de línea al momento de escribir renglones 
<p>Utilizamos la batidora para preparar la masa.<br>Después procedemos a agregar la sal.<p>
