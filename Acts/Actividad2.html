Explicación de un breve código de C en HTML

Código


    #include <stdio.h>

    int main ()
    {
        int edad;
        printf("¿Cuántos años tienes?");
        scanf("%d", &edad);

        if(edad>=18)
        {
            printf("Eres mayor de edad.\n");
        } else
        {
            printf("Eres menor de edad.\n");
        }
        
        return 0;
    }

Explicación del código

Adición de bibliotecas

Utilizamos #include para incluir la biblioteca requerida.
<stdio.h> es la biblioteca estándar utilizada para códigos en lenguaje C.

Función principal

La función principal es definida mediante int main.
Siendo int la manera para definir nuestra función y la palabra main la utilizada (proveniente del inglés) cuyo signifcado o traducción es "principal".

Declaración de variables

Para declarar una variable utilizamos el int y le damos el nombre de elección.
Esto nos da una vairable de tipo integer la cual no registra decimales.

Envío de texto a pantalla

Utilizamos la función printf() para poder mandar texto a pantalla.
Para que se respete el fromato utilizamos comillas dentro de los paréntesis para encerrar nuestro texto.

Obtención de información

Para poder registrar la información utilizamos la función scanf().
Para que esta registre la información como lo erquerimos utilizamos el & y d dentro de las comillas para que nos registre
un conjunto númerico del 0 al 9 dentro de la variable seleccionada.

IF & ELSE

Utilizamos la estrcutura if para poner una condición específica. En este caso la de que el número sea mayor o igual a 18.
Una vez que esto se verifica se deriva en dos posibles resultados, que la condición haya sido cumplida o que la condición no haya sido cumplida.
En caso se manda el mensaje "Eres mayor de edad" y en caso dos se manda "Eres menor de edad".
